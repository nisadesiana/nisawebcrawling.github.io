    Jadi, simpan di variable src untuk website target kemudian menentukan ada web yang akan diambil kali ini akan mengambil judul, title, dan isi pada web tersebut.

    Untuk dapat melakukan crawling data yang tepat sesuai dengan keinginan, terlebih dahulu harus melakukan inspeksi elemen dengan cara menekan tombol ctrl+i pada saat membuka website tersebut, lalu mencari tag dimana data yang kita inginkan berada.

    Artikel yang ingin kita crawl berada pada tag <a>

    soup.find(parameter) digunakan untuk mendapatkan satu tag html yang muncul pertama kali

    soup.findAll(parameter) digunakan untuk mendapatkan semua tag html tersebut`

    Untuk mendapatkan textnya, digunakan `.getText() pada objek BeautifulSoup.
