    Crawling

    Tools yang digunakan untuk Web Structure Mining sama dengan Web Mining klik disini , lalu ditambah bebera tools baru yaitu:

    1. Pandas : berfungsi untuk memuat file ke dalam tabel Virtual yang disimpan pada RAM. Pandas dapat mengolah suatu data dan mengolahnya seperti join, distinct, group by, agregasi, dan teknik seperti pada SQL.

    2. networkx : merupakan library Python untuk pembuatan, manipulasi, dan studi tentang struktur, dinamika, dan fungsi jaringan yang kompleks.

    3. matplotlib : merupakan suatu library paling banyak digunakan oleh data science untuk menyajikan datanya ke dalam visual yang lebih baik. Terdapat plot untuk menampilkan data secara 2D atau 3D.

    Tahap crawling beserta code

    • Import tools

    python import pandas as pd import requests from bs4 import BeautifulSoup import matplotlib.pyplot as plt import networkx as nx

    • Melakukan filtering link

    python def checkURL(url): if "www." in url: ind = url.index("www.")+4 url = url[ind:] if not "http" in url: url = "http://"+url if url[-1] == "/": url = url[:-1] return url Kondisi pertama: untuk menambahkan "http://" pada link dengan index awal "www."

    Kondisi kedua: untuk menambahkan "http://" pada link

    Kondisi ketiga: untuk mengecek "/" di akhir link atau link[-1]="/", maka ambil link dari index link[-1] sampai index awal link atau link=link[:-1].

    • Pengambilan seluruh link pada website

    ```python def getAllLinks(src): try: page = requests.get(src) soup = BeautifulSoup(page.content, 'html.parser')

      tags = soup.findAll("a")

  links = []
  for tag in tags:
      try:
          link = tag['href']
          if not link in links and 'http' in link:
              links.append(link)
      except KeyError:
          pass
  return links

    except: return list()

    • Crawl Data

    max_deep=Maksimal kedalaman, show=untuk menampilkan proses crwal, deep=0 merupakan kedealaman kedalam awal untuk memulai crawling, dan done[] merupakan list penampung link yang sudah di ambil

    ```python def crawl(url, max_deep, show=False, deep=0, done=[]): global edgelist deep += 1

      url = checkURL(url)

  if not url in done:
      links = getAllLinks(url)
      done.append(url)
      if show:
          if deep == 1:
              print(url)
          else:
              print("|", end="")
              for i in range(deep-1): print("--", end="")
              print("(%d)%s" %(len(links),url))

      for link in links:
          link = checkURL(link)
          edge = (url,link)
          if not edge in edgelist:
              edgelist.append(edge)
          if (deep != max_deep):
              crawl(link, max_deep, show, deep, done)

    `` kondisiif not url in done` dibuat untuk melakukan pengecekan agar link yang sudah pernah dicrawl tidak dimasukkan pada edgelist.
